# HyperCasual_Engeller

Unity ile geliştirdiğim hyper-casual tarzda bir engel prototipi. 
Proje, temel oyun mekaniği, engel davranışları ve oyuncu etkileşimi üzerine odaklanmaktadır.

## 🎯 Projenin Amacı
Basit ama akıcı bir gameplay deneyimi oluşturmak ve modüler engel sistemleri tasarlamak.

## 🛠 Kullanılan Teknolojiler
- Unity 6
- C#
- Prefab Sistemi
- Collider & Trigger Mekanikleri
- DOTween (animasyonlar)
- State Machine (oyun akışı yönetimi)

## ⚙️ Öne Çıkan Özellikler
- Modüler obstacle sistemi
- Dönen / hareketli / çarpışma bazlı engeller
- Prefab tabanlı yapı
- Temel zorluk dengesi denemeleri

## 🎮 Oynanış
Oyuncu engellerden kaçınarak ilerler. 
Engeller farklı davranış tiplerine sahiptir ve oyuncu refleksine dayalı bir deneyim sunar.

## 📂 Proje Yapısı
Assets/
- Scripts
- Prefabs
- Scenes

## 🚀 Geliştirme Notları
Bu proje, gameplay sistemleri ve engel tasarımı üzerine deneyim kazanmak amacıyla geliştirilmiştir.
